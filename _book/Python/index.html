Sublime Text 3 增加Python解释器支持

试验环境： 我的Sublime Text版本3.1.1

操作系统：Mac OS

参考链接

首先确保本地已可运行python命令

python3
Python 3.7.2 (default, Feb 18 2019, 14:58:37) 
[Clang 10.0.0 (clang-1000.11.45.5)] on darwin
Type "help", "copyright", "credits" or "license" for more information.

查看本地python3解释器所在位置

liuweizhendeMacBook-Pro-2:~ liuweizhen$ which a python3
/usr/local/bin/python3

// 也可以：
liuweizhendeMacBook-Pro-2:~ liuweizhen$ type -a python3
python3 is /usr/local/bin/python3

对sublime增加解释器支持 Sublime Text > Tools > Build System > New Build System

{
    "shell_cmd": "make"
}

替换成上面通过命令找到的python3解释器路径：

{
    "cmd": ["/usr/local/bin/python3"],
}

command+s保存，弹出弹框名字为untitled.sublime-build

可更改名字，但后缀.sublime-build不可改动

测试

新建hello.py

#!/usr/bin/python3

print("Hello world!")

command+b，下面控制台打印：

Hello world!
[Finished in 0.2s]
